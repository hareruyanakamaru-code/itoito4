# OGP画像 改善方針

> 作成日: 2026年4月20日
> 目的: SNSシェア時のクリック率向上・ブランド認知の強化

---

## 現状の課題

現在のOGP画像は `app/opengraph-image.tsx` でNext.jsのImageResponseを使い、テキストのみで動的生成されている。

**問題点：**
- 写真がなくテキストだけ → SNSのタイムラインで目に留まりにくい
- 体験の雰囲気（子どもの笑顔・ワクワク感）が伝わらない
- 他のサービス（Gifte!やいこーよ）は写真メインのOGPで差がついている
- 右下URLとカテゴリバッジが重なって表示される可能性

---

## 改善方針

### A. トップページ用OGP（ブランド用）

**方向性**: 子どもが体験を楽しんでいる写真 + ブランドロゴ + キャッチコピーの合成画像

**レイアウト案（1200x630）:**
```
┌──────────────────────────────────────────────┐
│                                              │
│   ┌─────────────┐  itoito イトイト           │
│   │             │                            │
│   │  体験写真    │  子どもの「やってみたい」を、│
│   │  （左半分）  │  本物の体験に変える。       │
│   │             │                            │
│   │             │  🌱 教師・職人・クリエイター  │
│   └─────────────┘     × 子ども               │
│                                              │
│  🍳 料理  🌿 自然  🎨 アート  🔍 探究        │
└──────────────────────────────────────────────┘
```

**実装方法（2パターン）:**

1. **推奨：Canva等で専用画像を作成 → 静的ファイルに差し替え**
   - `public/images/ogp-main.png`（1200x630）を作成
   - `app/opengraph-image.tsx` を削除し、静的ファイルを参照
   - メリット：写真を自由に配置、フォントも自由、品質が最も高い
   - はれるやさんの作業：Canvaで写真入りのOGP画像を1枚作る

2. **代替：動的生成を維持しつつ背景写真を合成**
   - 体験中の写真を背景に敷き、半透明オーバーレイ＋テキストを重ねる
   - `public/images/ogp-bg.jpg` に写真を置き、ImageResponseで合成
   - メリット：コードで管理できる
   - デメリット：フォント制約あり、写真のトリミング調整が難しい

### B. 体験個別ページ用OGP

**現状**: 体験画像をそのままOGPに使用（`exp.images?.[0]`）
**改善**: 体験写真 + タイトル + 価格 + 日時をオーバーレイ表示する動的OGPに変更

**レイアウト案（1200x630）:**
```
┌──────────────────────────────────────────────┐
│                                              │
│         体験写真（全面背景）                   │
│         ＋半透明グラデーション                  │
│                                              │
│  ┌─────────────────────────────────────────┐ │
│  │ 🌱 itoito                               │ │
│  │                                         │ │
│  │ 本格パン作り体験！                       │ │
│  │ プロのパン職人と一緒に焼きたてパンを作ろう │ │
│  │                                         │ │
│  │ 📅 5/10(土) 10:00  📍江東区  ¥3,000    │ │
│  └─────────────────────────────────────────┘ │
└──────────────────────────────────────────────┘
```

**実装**: `app/experiences/[id]/opengraph-image.tsx` を新規作成

## はれるやさんへのお願い

トップページ用OGP画像（1200x630px）をCanvaで作成してください。以下の要素を入れるのがおすすめです：

- 子どもが体験を楽しんでいる写真（できれば実際のitoito体験の写真。なければフリー素材でOK）
- 「itoito」のロゴ文字
- 「子どもの『やってみたい』を、本物の体験に変える。」のコピー
- 暖色系（アンバー・オレンジ）のトーンで統一

Canvaテンプレート検索ワード: 「OGP画像」「SNSシェア画像」「1200x630」

作成した画像を共有いただければ、すぐにサイトに反映します。
